# Objetivo do MWCode

MWCode é um sistema flexível de agentes de IA que opera em dois modos complementares: uso pessoal (Single) ou empresarial (Enterprise).

## Metas

- Dar ao usuário controle total sobre qual provedor/modelo usar
- Permitir trocar de provedor sem trocar de código (adapters unificados)
- Manter o sistema rodando mesmo com APIs grátis (via failover)
- 100% em português brasileiro
- Zero lock-in: funciona local (Ollama) ou na nuvem

## Usuários-alvo

1. **Dev solo** — quer chat direto com IA, sem custo (Single + Ollama/OpenRouter grátis)
2. **Empresa pequena** — 3-10 agentes com funções diferentes (dev, suporte, copywriter), precisa ver custos
3. **Enterprise** — múltiplos times, orçamento controlado, analytics
